MySQL Web Application

Web application for managing users, operating systems and installations. Built using Java 21, Hibernate ORM, JSP/Servlets and MySQL with full CRUD operations.

This page displays the list of all users stored in the database in a table format. The user can view available information and perform add, edit, or delete operations.

The operating systems page allows displaying all OS entries stored in the database. Each record can be modified or deleted, and new systems can be added.

This page displays all recorded installations, including the user, operating system, installation date, and associated device. Data is presented in a table with full management options.

The installation form allows selecting a user and an operating system from dynamic dropdown lists populated from the database. It also includes fields for installation date and device name.

The user interface of the application is implemented using JSP and HTML, designed to provide a simple and intuitive user experience. The layout is structured for quick and efficient access to all functionalities.
