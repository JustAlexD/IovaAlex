MySQL Web Application

Web application built with Java, Hibernate, JSP and MySQL.

Aceasta pagina afiseaza lista tuturor utilizatorilor existenti in baza de date sub forma unui tabel. Utilizatorul poate vizualiza informatiile disponibile si poate accesa actiunile de adaugare, modificare sau stergere a inregistrarilor.

Pagina dedicata sistemelor de operare permite afisarea tuturor sistemelor existente in baza de date. Fiecare inregistrare poate fi modificata sau stearsa, iar utilizatorul are posibilitatea de a adauga sisteme noi.

Aceasta pagina afiseaza toate instalarile realizate, incluzand utilizatorul, sistemul de operare, data instalarii si dispozitivul asociat. Informatiile sunt prezentate intr-un tabel.

Formularul de adaugare a unei instalari permite selectarea unui utilizator si a unui sistem de operare din liste derulante dinamice, populate automat din baza de date.

Interfata utilizator a aplicatiei este realizata folosind JSP si HTML, avand ca scop oferirea unei experiente de utilizare simple si intuitive.
